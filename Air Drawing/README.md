# Air Drawing Canvas Using MediaPipe

Draw on the screen using hand gestures and a webcam.

## Features

* Air drawing using index finger
* Gesture-based mode switching
* Clear canvas gesture
* Real-time tracking

## Technologies

* Python
* OpenCV
* MediaPipe
* NumPy

## Landmarks Used

Index Finger Tip : 8

Middle Finger Tip : 12

## Gestures

Drawing Mode

Index finger up

Selection Mode

Index finger + Middle finger up

## Installation

pip install mediapipe opencv-python numpy

## Run

python main.py
