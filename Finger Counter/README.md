# Finger Counter Using MediaPipe

A real-time hand gesture project that counts the number of fingers shown to the webcam.

## Features

* Real-time hand tracking
* Detects 0 to 5 fingers
* Uses MediaPipe hand landmarks
* Displays finger count on screen

## Technologies

* Python
* OpenCV
* MediaPipe

## Landmarks Used

Thumb Tip : 4

Index Tip : 8

Middle Tip : 12

Ring Tip : 16

Pinky Tip : 20

## Installation

pip install mediapipe opencv-python

## Run

python main.py

## Learning Outcomes

* MediaPipe hand landmarks
* Coordinate systems
* Landmark indexing
* Gesture recognition basics
