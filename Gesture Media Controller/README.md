# 🎵 Gesture Media Controller

Control music playback and system volume using hand gestures and MediaPipe.

## Features

* Play / Pause music
* Next Track
* Previous Track
* Volume Up
* Volume Down
* Real-time hand tracking

## Technologies

* Python
* OpenCV
* MediaPipe
* PyAutoGUI

## Installation

pip install mediapipe opencv-python pyautogui

## Run

python main.py

## Gestures

👍 Thumbs Up → Play / Pause

☝️ Index Finger → Previous Track

✌️ Index + Middle → Next Track

🖐️ All Fingers Open → Volume Up

🤏 Thumb + Index Pinch → Volume Down
